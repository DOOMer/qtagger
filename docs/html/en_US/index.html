qTagger - a simple editor id3-tags

qTagger is a simple application designed editing id3-tag mp3-files. In addition to the application includes a function automated conversion of tags from different national character sets to UTF-8.

qTagger is running on GNU/Linux and Microsoft Windows (XP/Vista/Seven)

To work with the id3-tag library is used TagLib, for GUI - Qt Development Framevork.

Using

Main oepratsii

Adding files

You can add files in several ways:

  • through the main menu - File->Add File.
  • clicking Add file on the toolbar.
  • use of Add file in the shortcut menu, the list of files.

Files are always added to the end of the list.

Adding directories

File directory can be added in several ways:

  • through the main menu - File->Add directory.
  • clicking Add directory on the toolbar.
  • use of Add directory in the shortcut menu, the list of files.

If the selected directory are nested subdirectories their content (mp3-files) will also be added to the program. Files are always added to the end of the list.

Convert tags in UTF-8

To convert the tags to the files from the local 8-bit encoding to utf-8 should select the files you need from the list and then click To Unicode on the toolbar. This can be done by selecting the main menu File->To Unicode, or the same name from the context menu list of files.

To select multiple files, press and hold Ctrl and using the mouse, choose the files you want.

Delete selected files from the list

Delete selected files can be produced by several sposoboami:

  • through the main menu - File -> Delete .
  • clicking Delete on the toolbar.
  • Use of Delete in the shortcut menu, the list of files.

To select multiple files, press and hold Ctrl and using the mouse, choose the files you want.

To remove all files from the list, click Clear on toolbar. Or select the appropriate item in the main (or context) menu.

Setting

Preset window can be accessed by clicking Nastryoka on the toolbar or selecting an item from the main menu.

The program provides only 2 custom parameter - Encoding and Toolbar . The first one selects the necessary initial encoding to convert non-UTF tags. The second - allows you to customize the look of the buttons on the toolbar - show them the text, icons or together with the text.
