qTagger -- простой редактор id3-тегов

qTagger является простым приложением, предназначенным для редактирования id3-тегов в mp3-файлах. Кроме этого в приложении реализована функция автоматизированного конвертирования тегов из различных национальных кодировок в UTF-8.

qTagger работает в операционных системах GNU/Linux и Microsoft Windows (XP/Vista/Seven)

Для работы с id3-тегами используется библиотека TagLib, для графического интерфейса -- Qt Development Framevork.

Интерфейс программы

Главное окно

Главное окно qTagger состоит из 3 основных частей:

  • Главное меню
  • Панель инструментов
  • Список загруженных файлов
  • Область редактирования тегов

Главное меню -- позволяет выполнять основные операции, предусмотренные в программе. Большая часть пунктов также продублирована на панели инструментов и в контекстном меню списка файлов.

Описание пунтов меню Файл .

  • Добавить каталог -- добавление каталога с файлами (включая также все вложенные каталоги)
  • Добавить файл -- добавление одного или нескольких файлов
  • Конвертировать -- конвертирование тегов ввыбранных файлов в кодировку UTF-8
  • Удалить -- удаление выбранных файлов из списка
  • Очистить -- осчистка всего списка
  • Настройки -- окно настроек программы
  • Выход -- выход из программы

Описание пунтов меню Инфо.

  • Справка -- отображение справочной системы (в HTML-формате) программы
  • О программе -- отображение информации о qTagger
  • О Qt -- отображение информации о Qt Development Framevork

Панель инструментов -- позволяет выполнять основные операции, предусмотренные в программе. Большая часть кнопок также продублирована в основном меню и в контекстном меню списка файлов.

Описание кнопок панели инструментов.

  • Добавить каталог -- добавление каталога с файлами (включая также все вложенные каталоги)
  • Добавить файл -- добавление одного или нескольких файлов
  • Конвертировать -- конвертирование тегов ввыбранных файлов в кодировку UTF-8
  • Удалить -- удаление выбранных файлов из списка
  • Очистить -- осчистка всего списка
  • Настройки -- окно настроек программы
  • Справка -- отображение справочной системы (в HTML-формате) программы

Список загруженных файлов -- предназначена для отображения списка файлов и операций над ними. Позволяет выбирать один или несколько файлов. Под списком файлов находится кнопка быстрого выбора/снятия выбора всех файлов в списке.

При клике по списку файлов правой кнопкой мыши появляется контекстное меню, в котором дублируются пункты основного меню и кнопки панели инструментов, относящиеся к операциям с файлами.

Область редактирования тегов -- состоит из нескольких полей ввода для заполнения основных id3-тегов. Здесь же расположены две кнопки:.

  • Сохранить -- сохранение внесённых изменений в выбранный файл
  • отмена -- сброс всех несохранённых изменений в выбранном файле

Работа с программой

Основные оепрации

Добавление файлов

Файлы можно добавлять несколькими способами:

  • Через главное меню -- Файл -> Добавить файл.
  • Нажатием на кнопку Добавить файл на панели инструментов.
  • При помощи пункта Добавить файл в контекстном меню списка файлов.

Файлы всегда добавляются в конец списка.

Добавление файлов

Каталоги с файлами можно добавлять несколькими способами:

  • Через главное меню -- Файл -> Добавить каталог.
  • Нажатием на кнопку Добавить каталог на панели инструментов.
  • При помощи пункта Добавить каталог в контекстном меню списка файлов.

Если в выбранном каталоге существуют вложенные подкаталоги, то их содержимое (mp3-файлы) будут также добавлены в программу. Файлы всегда добавляются в конец списка.

Конвертирование тегов в кодировку UTF-8

Чтобы конвертировать теги в файлах из локальной 8-битной кодировки в utf-8 необходимо выделить нужные вам файлы в списке и затем нажать кнопку конвертировать на панели инструментов. Также это можно сделать выбрав пункт главного меню Файл -> Конвертировать, либо одноимённый пункт контекстного меню списка файлов.

Для выбора нескольких файлов нажмите и удерживайте клавишу Ctrl и при помощи мыши выбирайте нужные файлы.

Удаление выбранных файлов из списка

Удаление выбранных файлов можно произвести несколькими способоами:

  • Через главное меню -- Файл -> Удалить.
  • Нажатием на кнопку Удалить на панели инструментов.
  • При помощи пункта Удалить в контекстном меню списка файлов.

Для выбора нескольких файлов нажмите и удерживайте клавишу Ctrl и при помощи мыши выбирайте нужные файлы.

Чтобы удалить все файлы из списка, нажмите кнопку Очистить на панели инструментов. Или выберите соответствующий пункт в главном (или контекстном) меню.

Настройка программы

Окно настроки можно вызвать нажав кнопку Настрйока на панели инструментов или выбрав соответствующий пункт в главном меню.

В программе предусмотрено всего 2 настраиваемых параметра -- Кодировка и Панель инструментов. Первый из них осуществляет выбор необходимой исходной кодировки для конвертирования не-UTF тегов. Второй -- позволяет настроить внешний вид кнопок на панели инструментов -- отображать на них текст, иконки или иконки вместе с тектом.
